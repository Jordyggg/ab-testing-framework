# A/B Testing Framework

**Author:** Jordy Garzón  
**Contact:** garzonjordy@hotmail.com

## Overview

Statistical framework for designing, running, and analyzing A/B tests. Includes sample size calculation and significance testing.

## Technologies

- Python
- SciPy / Statsmodels
- Pandas
- Plotly Dash
- Jupyter

## Features

- Professional implementation
- Clean, documented code
- Best practices
- Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
